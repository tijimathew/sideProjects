BlackJack - Simple text based game
------------------------------------
x The game uses a deck of 52 Cards (https://en.wikipedia.org/wiki/Standard_52-card_deck)
- Each card has a value 
  x K, Q, J -> Face Cards -> value = 10
  - A -> Ace -> value = 1 or 11
  - 2 through 10 -> number cards -> value = number on card
- There are 4 suites of each card in a deck of 52 cards.


The game needs to have one player versus an automated dealer.
The player can stand or hit.
The player must be able to pick their betting amount.
You need to keep track of the player's total money.
You need to alert the player of wins, losses, or busts, etc...
Option for multiple players
Option to play with multiple Decks
Option to Double-Down
OPtion to do card splits
